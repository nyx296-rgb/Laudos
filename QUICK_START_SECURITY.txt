🚀 PRIMEIRAS AÇÕES (FAÇA AGORA):

1. COPIAR ARQUIVO DE CONFIGURAÇÃO:
   
   cp .env.example .env

2. EDITAR .env (abra em notepad):
   
   Busque por: ADMIN_PASSWORD=ChangeMe@123
   Mude para: ADMIN_PASSWORD=SuaSenhaForte@123

3. REINICIAR SERVIDOR (opcional, já reiniciado):
   
   Ctrl+C no terminal (se estiver rodando)
   python run_server.py

4. TESTAR:
   
   - Abra http://localhost:5000
   - Login com: admin / SuaSenhaForte@123
   - Deve funcionar!
