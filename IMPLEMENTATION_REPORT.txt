OPÇÃO 1: Configuração Simples (Testes)

2. Inicie servidor: python run_server.py
3. Login: admin / ChangeMe@123
4. Mude a senha no perfil do admin

OPÇÃO 2: Configuração Segura (Produção)
────────────────────────────────────────────────────────────────────────────
1. Gere chaves seguras:
   python -c "import secrets; print(secrets.token_hex(32))"
2. Crie .env com valores:
   SECRET_KEY=<gerar novo>
   ADMIN_PASSWORD=<senha forte única>
3. Não commit .env no git
4. Inicie servidor: python run_server.py

MIGRAÇÃO DE DADOS EXISTENTES:
────────────────────────────────────────────────────────────────────────────
Se você já tinha usuários:
1. Backup do banco: cp laudos.db laudos.db.backup
2. Execute: python migrate_passwords.py
3. Valide: SELECT password FROM users (deve conter $2b$)
